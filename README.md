# Jack-Track

Spend Track for Jack — a mobile-first monthly expense tracker.

Built with React 19, Vite, Tailwind v4, framer-motion, and recharts.

## Development

```sh
npm install
npm run dev      # start the Vite dev server
npm run build    # production build into dist/
npm run lint     # ESLint
```

See [CLAUDE.md](./CLAUDE.md) for architecture notes.
